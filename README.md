# Squat Kinematics Analysis Pipeline📊

A MATLAB pipeline designed to import, process, and analyze full lower-limb kinematic data during squat movements. This script takes 3D motion data (specifically .mot files generated by OpenCap/OpenSim) and extracts clinically relevant variables, including joint ranges of motion, peak velocities, accelerations, and a Symmetry Index (SI).

## Features
* Data Interpolation: Automatically detects and fills missing data points (NaN values) using linear interpolation, which is highly useful for smartphone-based motion tracking.
* Signal Filtering: Applies a 4th-order zero-lag Butterworth low-pass filter (Cutoff frequency: 6 Hz) to smooth the raw data without phase shifts.
* Phase Detection: Automatically identifies the bottom phase of the squat (maximum knee flexion) to segment the descending and ascending phases.
* Derivatives Calculation: Computes angular velocity and angular acceleration for the hip, knee, and ankle joints.
* Symmetry Index (SI): Compares left vs. right kinematics to detect mild or significant asymmetries during the movement.
* Data Export: Automatically generates comprehensive visualization plots and exports the numerical results into a .csv file for further analysis.

## Data Source
* System: OpenCap / OpenSim IK output
* File format: .mot (Numeric Matrix)
* Sampling Rate: 60 Hz

## How to Use
1. Ensure your .mot file is loaded into the MATLAB workspace as a variable named Squat.
2. Run the script.
3. The script will output the phase timings in the command window, generate a 3x3 figure with all kinematic plots, and save a squat_kinematics_results.csv file in your current directory.
